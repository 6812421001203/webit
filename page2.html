☁️ 2. Cloud & Servers (เซิร์ฟเวอร์ฟรี)

สำหรับคนที่อยากลองวิชาสาย System/DevOps อยากมีเซิร์ฟเวอร์ส่วนตัวไว้รันบอท, เปิดเซิร์ฟเกม, หรือโฮสต์เว็บจริง

สิ่งที่ได้รับ:

  • Microsoft Azure:
    เครดิตฟรี $100 ต่อปี + เข้าถึงบริการฟรีอีกกว่า 25 รายการ (ไม่ต้องใช้บัตรเครดิต) เหมาะสำหรับเรียนรู้ Cloud ระดับองค์กร
  • DigitalOcean:
    เครดิตฟรี $200 (อายุใช้งาน 1 ปี) ใช้สร้าง Droplet (VPS) ได้ง่ายและเร็วมาก เหมาะสำหรับทำ Web Server หรือ Game Server (Minecraft/FiveM)
  • Heroku:
    เครดิตสำหรับนักเรียน (ประมาณ $13/เดือน นาน 24 เดือน) แพลตฟอร์มยอดฮิตสำหรับ Deploy App ภาษา Node.js, Python, Ruby แบบง่ายๆ
